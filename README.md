# discque-hearto-pulse
A Pulsing Heart Disco Floor in POV-Ray

# rendering
Run povray with `povray discque-hearto-pulse.pov anim.ini` or for a still use `povray discque-hearto-pulse.pov +H1080 +W1920 +A +AM2 +Q11`
